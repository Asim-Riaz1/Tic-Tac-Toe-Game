## Tic Tac Toe (C++)

A simple console-based Tic Tac Toe game written in C++ as part of learning core language concepts.

### Features
- 3x3 board represented using arrays
- Player vs computer gameplay
- Randomized computer moves
- Win and tie condition checks
- Modular design using functions

### Purpose
Practice C++ fundamentals including arrays, functions, loops, conditionals, and basic input/output.
